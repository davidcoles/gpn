First time use:
  • Log in via SSO using the link above. The page background should go from light blue to yellow.
  • If you don't already have WireGuard installed, download it from the link above.
  • Download a WireGuard configuration file from the "config" link above.
  • In WireGuard use "Import Tunnel(s) from File" to load the config file you have just downloaded.
  • Activate the new tunnel in WireGuard - the page background should change to light green after a moment.
  • You should now be able to access internal systems to which you have access.
  • Let support know that your device has been set up so that they can do some admin.
  • Bookmark this page so that you can return to it for diagnostic info if your VPN doesn't seem to work.
